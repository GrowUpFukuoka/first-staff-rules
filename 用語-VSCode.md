# VSCode について
* ソースコードエディタです。
* 多様なプログラミング言語や記法に対応しています。
* シンタックスハイライトやコード補完があります。
* Gitのフロントエンドとしても使えます。

## 対応している環境
* Windows
* Linux
* macOS

## 料金
無料です。

## 安全性
* オープンソースです。
* Microsoft社が開発しています。
